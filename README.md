A PowerShell-themed multiple-choice quiz designed to showcase scripting skills, JSON-driven logic, and defensive coding practices. The project includes both a local PowerShell version and a web version playable directly in the browser.



Project Structure

powershell-skills-game/

├─ powershell/

│  ├─ PowerShell-MCQ-Game.ps1   # Local text-based game

│  └─ questions.json             # JSON questions for PowerShell game

├─ web/

│  ├─ index.html                 # Web version HTML

│  ├─ script.js                  # Web version JS logic

│  └─ CSS panel included in CodePen

└─ README.md



Versions

1️⃣ PowerShell (Local)



Runs in PowerShell console



Text-based multiple-choice quiz



Reads questions from questions.json



Shows final score after completion



2️⃣ Web Version (Browser)



HTML + CSS + JS



Terminal-style interface (green-on-black for PowerShell feel)



Fully playable in browser (CodePen-ready)



Self-contained (no external JSON needed)



Live link can be shared with recruiters



Tech Highlights



PowerShell scripting: loops, input validation, scoring



JSON-driven question structure



Defensive coding to avoid errors and invalid input



Web version: interactive HTML + JS, terminal-style CSS



How to Play

PowerShell



Open PowerShell-MCQ-Game.ps1 in PowerShell



Run the script



Follow prompts, type your answer number, press Enter



Score displayed at the end



Web (Browser)



Open index.html in browser or copy CodePen link



Click the answer buttons



Score displayed at the end
